This AI-Powered Book Recommender helps you find your next read by understanding the content and feeling of books. Simply enter a description of the plot or theme you're looking for, choose between Fiction and Non-Fiction, and select the desired emotional tone. The system semantic search to retrieve similar books from a vector database. The recommendations are displayed on an interactive frontend made using Gradio.

Dataset used : https://www.kaggle.com/datasets/mdhamani/goodreads-books-100k/data

Score after fine tuning DeBERTa for classification between Fiction and Nonfiction :
Accuracy	F1	      Precision	Recall
